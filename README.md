 Project Overview

TaskFlow is a  full-stack To-Do management web application** built using the MERN stack.  
It allows users to **securely register, log in, and manage their daily tasks** with full CRUD functionality.

The application uses **JWT-based authentication** and **protected routes**, ensuring that only authenticated users can access and manage their personal To-Do items.
